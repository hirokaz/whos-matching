本仕様書のベースペルソナは女性だが、**ゲーム本体では男性プレイヤーも遊べるよう主人公の性別を選択可能とする**（詳細は `character-spec.md` 参照）。

## ■ Claudeへの適用指示

シナリオ生成時は以下を必ず守ること：

1. 主人公は常識的な行動を取ること
2. すべての選択肢は現実的にあり得るものにする
3. 無理な誘導は禁止
4. 恋愛 → 仕事 → 恐怖の流れを維持する
5. 違和感は徐々に積み上げる
6. 恐怖は説明せず「気づかせる」
7. プレイヤーが自然に判断ミスをする構造にする
